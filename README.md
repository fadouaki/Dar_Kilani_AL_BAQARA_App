# دار الكيلاني

## Description

This application collects the "athman" (parts) of Surah Al-Baqara that were published on the Facebook page of "Dar El Kilani for the memorization of the Holy Quran with the rules of Tajweed and the reasons for revelation". The page is dedicated to memorizing the Holy Quran with the rules of Tajweed, explaining scientific and legal texts, the reasons for revelation, and teaching the Islamic sciences. May Allah make this work in the balance of the good deeds of everyone who contributed to it and benefited from it.

## How to Test

You can test the application by installing the debug APK file located at:

`app/build/outputs/apk/debug/app-debug.apk`

## How to Build

To build the application from source, you can use Android Studio or the following Gradle command:

```bash
./gradlew assembleDebug
```

This will generate the debug APK in the same location as mentioned above.
